## [0.9.0-dev] - 2025-09-10
### Added
- Поддержка «получасового BREAK» — ожидание до 30-й минуты при раннем срабатывании второго кулачка (27–29 мин).
- Логика выхода из BREAK теперь универсальна (00 или 30 минута).
- Расширенные условия нормального срабатывания микрика (27–29 мин вместо только 29).

### Changed
- Рефакторинг FSM стрелки: упрощена структура `case` и условия переходов.
- Упорядочены `#include` в модулях, устранены конфликты имён с библиотекой `<WiFi.h>`.
- Перенос функций по файлам для лучшей модульности.

### Fixed
- Исправлен фальстарт при переходе из BREAK на 59-й минуте.
- Устранена рекурсивная петля `#include` и дублирование определений функций.
- Очистка кэша Arduino IDE для устранения «призраков» старых `.o` файлов.
